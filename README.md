# LightBoard / 轻白板

一个基于 WPF `InkCanvas` 的极简触摸绘图板，专为触控屏/大屏设计。支持多指手势、撤销重做、墨迹保存与导出。

> Author: KaiHuaDou

## 功能

- 多色画笔与多种笔触粗细
- 荧光笔模式
- 点擦除 / 笔画擦除 / 选择工具
- 撤销 / 重做
- 打开 / 保存 `*.isf` Windows 墨迹文件
- 导出画布为 `*.png`
- 单实例运行
- 崩溃时自动备份当前墨迹

## 触摸手势

| 手指数量 | 状态 | 说明 |
|---------|------|------|
| 1 | 绘图 | 单指轻触即可书写或绘制 |
| 2（近距离）| 平移 + 缩放 | 双指移动平移，张合缩放 |
| 3 ~ 4（近距离）| 平移 | 多指拖动平移画布 |
| ≥ 5（近距离）| 橡皮擦 | 多指用作大面积橡皮擦 |
| ≥ 2（远距离）| 多人绘制 | 多人在大屏两侧同时书写 |

> 近距离/远距离由窗口宽度的 60% 作为阈值判断。

## 系统要求

- Windows 7 或更新版本
- .NET 6.0 桌面运行时或更新版本

## 构建

```bash
dotnet publish -p:PublishProfile=FolderProfile -f net6.0-windows
``````
